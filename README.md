# Roulette

- TypeScript
- Three.js
- Blender
- Sketch

## Build

### Development

Run `vite` on http://localhost:3000 

```bash
$ bun run dev
```

### Production

Build a production bundle to `./dist`

```bash
$ bun run build
```
